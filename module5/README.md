# Module 5: Boundary Structure Recovery (Experimental)

> **Status: Experimental.** This module has not been fully validated and its results should be treated as preliminary. The code runs and produces output, but the design, parameter choices, and interpretation have not undergone the same scrutiny as Modules 1 to 4.

## Purpose

Modules 1 to 4 validate conditional logit for estimating individual-level partner preferences. Module 5 asks a different question: whether clogit can also recover group-level boundary structures, meaning the full matrix of pairwise affinities and aversions between ancestry groups, and how it compares to the simulation-based approach used in Paper 2 of the dissertation.

Paper 2 estimates social boundaries by comparing observed union counts to null distributions generated by Monte Carlo simulation under geographic opportunity constraints, then summarising via log-linear models. Module 5 tests whether this pipeline recovers known boundary structures, and compares it to a clogit specification with K² group-pair interaction terms.

## Design

The module generates a population of N agents from K = 5 ancestry groups with known pairwise preference parameters (an asymmetric K × K matrix of affinities and aversions). Agents are placed on a 1D spatial grid with group-specific geographic distributions. Three estimation approaches are compared:

1. **Crude ratio**: log(observed / expected) for each cell in the union matrix
2. **Log-linear three-way interaction**: WS × MS × WM interaction from the contingency table
3. **Clogit with K² interactions**: individual-level discrete choice model with group-pair fixed effects

Each approach is tested under bilateral matching at three selectivity levels (25%, 50%, 75% one-sided acceptance rate) across 100 Monte Carlo repetitions.

## Running

```bash
cd module5
Rscript module5_run.R              # full run (~1 to 2 hours)
Rscript module5_run.R quick        # sanity check (~5 min, 5 MC reps)
```

Requires the same R packages as the main modules plus `nnet` for multinomial logit.

## Known Limitations

- The geographic structure is 1D (for tractability) and may not capture spatial opportunity as it operates in the real data.
- The K = 5 setup is much smaller than Paper 2's 50 groups. Scaling behaviour is unknown.
- The true preference matrix is hand-specified rather than estimated from empirical patterns.
- The endogamy absorption effect identified in preliminary results needs further investigation.
- Results have not been cross-checked against analytical expectations or alternative implementations.

## Preliminary Findings

Clogit with K² interactions recovers the off-diagonal boundary structure well (r = 0.87 to 0.94) across all bilateral conditions, while the simulation pipeline shows poor off-diagonal recovery (r = -0.1 to -0.8). The simulation pipeline performs reasonably on the diagonal (endogamy detection) but struggles with cross-group affinities due to an endogamy absorption effect. These findings are preliminary and require further validation before being cited.
